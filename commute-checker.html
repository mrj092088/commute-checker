ⓘ How to get a free Google Maps API key

🔑 One-time Setup (~5 min)

  1. Go to console.cloud.google.com
  2. Create a project (e.g. "Commute")
  3. Enable: Routes API
  4. Go to APIs & Services → Credentials
  5. Create an API key — restrict it to your IP or HTTP referrer
  6. Paste it above and tap Save

* Free tier: 10,000 route requests/month. Well within personal use.
